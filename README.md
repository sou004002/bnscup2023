# 概要
タイトル：「餌をやる。そしてごみをすくう。」
バージョン：1.00
ジャンル：育成耐久ゲーム
プレイ時間：3～5分
制作ソフト：C++フレームワーク「Siv3D」
***
水槽の中の生き物を育成するゲームです。
餌をやると経験値が増え、レベルが上がります。
ごみを片付けないと、HPが減ってしまいます。


# 操作方法
左クリック：餌をやる・ごみをすくう
右クリック：餌モード、ごみモードの切り替え
escキー：終了


# 起動方法
「餌をやる。そしてごみをすくう。」フォルダを解凍し、exeファイルを実行してください。
